🔧 技术实现

1. 核心函数：buildDiaryPrompt()

async buildDiaryPrompt(classifiedData, config) { // 1. 构建基础数据上下文 const dataContext = \`今日浏览数据：\${JSON.stringify(classifiedData)}\`; // 2. 获取前一天日记（如果启用对比） let comparisonContext = ''; if (config.enableDiaryComparison) { const previousDiary = await this.getPreviousDiary(); if (previousDiary) { comparisonContext = \` 前一天日记：\${previousDiary} 对比要求：分析行为模式变化\`; } } // 3. 组合最终提示词 return \`\${dataContext}\${comparisonContext}\${defaultPrompt}\`; }

2. 获取前一天日记：getPreviousDiary()

async getPreviousDiary() { // 计算前一天日期 const currentDate = new Date(this.currentDate); currentDate.setDate(currentDate.getDate() - 1); const previousDate = currentDate.toISOString().split('T')[0]; // 从数据库获取 const diaryKey = \`diary-\${previousDate}\`; const result = await window.dbWrapper.get(diaryKey); return result[diaryKey] || null; }

🧠 默认提示词解析

认知模式分析框架

四层思想阶梯：

  • 阶梯一 - 本能与冲动：刷短视频、看八卦
  • 阶梯二 - 规则与角色：工作任务、学习技能
  • 阶梯三 - 自我与反思：哲学思考、自我分析
  • 阶梯四 - 整合与超越：开源贡献、系统思考

分析风格特点

  • 🎯 客观精确：冷静、克制的分析语言
  • 🔍 证据驱动：每个结论都标注具体证据
  • ⚡ 犀利点评：可以直接指出问题，监督用户
  • 📝 纯文本：禁止Markdown，确保输出纯净

🎨 自定义提示词

✅ 可以自定义！

在设置页面的"自定义日记提示词"中输入你的要求：

示例1：幽默风格

请用幽默风趣的语言分析我的浏览行为， 多使用表情符号，重点关注学习进展 和时间管理。

示例2：专业分析

请从心理学角度分析我的认知模式， 识别潜在的认知偏差，提供科学的 改进建议。

🔄 自定义与对比的结合

当你设置了自定义提示词后，系统会这样组合：

最终提示词 = 今日数据 + 前一天日记 + 你的自定义要求

📈 对比分析示例

输入：

前一天日记

主导模式：规则与角色驱动。大量工作任务处理。

次要模式：本能冲动零星出现。

能量流动：任务执行→本能冲动的泄压模式。

今日浏览数据

自我反思类：20条记录

哲学思辨类：15条记录

娱乐内容：5条记录（减少60%）

输出：

认知模式快报 - 2025-01-04

主导模式：自我与反思驱动。相比昨日的任务导向，今日更多投入个人成长。

模式进化：从昨日的"规则-本能"二元模式，跃迁至"反思-整合"的高阶模式。

行为变化：娱乐时间减少60%，学习深度提升，出现系统性思考迹象。

对比洞察：昨日的工作压力"泄压"模式转变为今日的主动"跃迁"模式。

⚙️ 如何启用

  1. 打开设置页面：点击右上角设置按钮
  2. 找到日记生成设置：滚动到"📝 日记生成设置"部分
  3. 启用对比功能：勾选"启用日记对比（与前一天的日记进行对比分析）"
  4. 可选自定义：在"自定义日记提示词"中输入你的要求（留空使用默认）
  5. 保存设置：点击"保存设置"按钮

💡 使用建议

🔄 连续使用

至少连续使用3-7天才能看到明显的对比效果和认知模式变化趋势。

⏰ 固定时间

尽量在每天相同时间生成日记，确保对比分析的一致性和有效性。

📊 关注趋势

重点关注认知模式的长期趋势，而不是单日的偶然变化。

🎯 目标导向

设定认知升级目标，通过对比分析追踪自己的进步情况。

🚀 开始你的认知进化之旅
